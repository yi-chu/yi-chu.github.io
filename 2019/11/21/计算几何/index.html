叉积

实现
应用
  1. 判断向量A,B的方向关系
    若Cross(A,B)>0,B在A的逆时针方向
    若Cross(A,B)<0,B在A的顺时针方向
    若Cross(A,B)==0,B与A共线 可能同向可能反向
  2. 计算两向量构成的平行四边形的有向面积及三角形面积 这里计算的是向量AB,AC构成的平行四边形的有向面积(也就是有正负的面积)
    三角形面积就是平行四边形面积除二。
    根据具体情况可选择取绝对值。
  3. 向量旋转
  4. 求单位法向量
  5. 检查两个向量是否共线
