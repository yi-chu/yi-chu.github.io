树的直径和树的重心

为了更好的了解树这种数据结构，这次就来学习一些树的基本特征。

树的直径

树的直径指树上距离最远的两点间的距离,换句话说就是树上的最长链。

树的直径的性质

  1. 直径两端点一定是两个叶子节点。

  2. 距离任意一个点最远的点一定是直径的其中一个端点。

  3. 用一条边把两棵树连接，新得到的树的直径的两个端点必然是这两棵树连接前的直径的端点其中之二。

牛妹的苹果树
需要st表欧拉序O(1)询问lca加快查询。

  1. 对于一棵树，如果在一个点上接一个叶子节点，那么最多会改变直径的一个端点。

  2. 若一棵树存在多条直径，那么这些直径交于一点且交点是这些直径的中点。

树的直径的中点的性质

  1. 在给定的树上找一个点使得树上任意一点到这个点的距离的最大值最小，这个点就是直径中点。
    且这个距离就是直径的长度/2向上取整。很容易用反证法证明其正确性。

  2. 如果现在要让你在树找两个点，使得树上任意一个点到这两个点的距离中较小的哪个距离的最大值最小。
    那么这两个点就是把这颗树从直径中点处断开，在两颗子树上取各自的直径的中点。

树的重心

树的重心也叫树的质心。找到一个点,其所有的子树中最大的子树节点数最少,那么这个点就是这棵树的重心,删去重心后，生成的多棵树尽可能平衡。

定义： 树的重心也叫树的质心。对于一棵树n个节点的无根树，找到一个点，使得把树变成以该点为根的有根树时，最大子树的结点数最小。换句话说，删除这个点后最大连通块（一定是树）的结点数最小。

树的重心的性质

  1. 树中所有点到某个点的距离和中，到重心的距离和是最小的，如果有两个距离和，他们的距离和一样。
  2. 把两棵树通过一条边相连，新的树的重心在原来两棵树重心的连线上。
  3. 一棵树添加或者删除一个节点，树的重心最多只移动一条边的位置。
  4. 一棵树最多有两个重心，且相邻。
  5. 一棵树的重心的最大子树的size不超过总结点的1/2。
