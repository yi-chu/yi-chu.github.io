什么是斯坦纳树

斯坦纳树的问题要从1643年法国业余数学家费马在他所写的一本关于求极值的书中提出的一个问题说起
这个问题被称为费马问题对费马问题不感兴趣可以点我跳过>.<

1643年，在一封写给意大利数学家和物理学家托里拆利（Evangelista Torricelli,1608–1647)的私人信件中，费马提出了下面这个极富挑战性和趣味性的几何难题，请求托里拆利帮忙解答（也有一种说法是费马本人实际上已经找到了这个问题的答案，他是为了挑战托里拆利才写信向他“请教”的）很符合费马的风格：
给定不在一条直线上的三个点 A,B,C，求平面上到这三个点的距离之和最短的点的位置。

托里拆利成功地解决了费马的问题。他给出的答案是：
对 △ABC 三条边的张角都等于120°，即满足∠APB =∠BPC =∠CPA = 120°的点 P（如下图所示）就是到点 A,B,C 的距离之和最小的点。

后来人们就把平面上到一个三角形的三个顶点 A,B,C 距离之和最小的点称为△ABC的费马-托里拆利点（Fermat-Torricelli point），也简称为费马点（Fermat point）或托里拆利点（Torricelli point）。

费马问题是我初中习题册上全等三角形那章的一道题，我印象很深刻。

要用几何方法解决费马问题，一种思想是把问题中的三条线段 PA, PB, PC“加”在一起或者说拼接在一起，最好是把它们拼接成连接两个定点的一条折线，这样一来，因为两点之间直线最短，就能很快地确定 PA + PB + PC 的最小值。利用旋转变换能成功地把费马问题中的三条线段以一种非常自然的方式“加到一起”。

只要把△BPC绕点B旋转60°（如上图所示），设点P转到了点P’，点C转到了点C’，于是就有
PC = P’C’， PB = PP’ （因为 △PBP’ 是等边三角形）
因此就有
PA + PB + PC = PA + PP’ + P’C’
上式的右边是连接点A和点C’的一段折线的距离，它一定大于或等于线段AC’的长度，所以我们就得到了不等式：
PA + PB + PC = PA + PP’ + P’C’ ≥ AC’
显然，如果上面的不等式能取到等号，那么这时候的点P就是到点 A, B, C 距离之和最小的点，也就是费马点。

斯坦纳树问题是组合优化问题，与最小生成树相似，是最短网络的一种。最小生成树是在给定的点集和边中寻求最短网络使所有点连通。而最小斯坦纳树允许在给定点外增加额外的点，使生成的最短网络开销最小。（ 个人理解:感觉最短路和最小生成树都是最小斯坦纳树中的一些特殊情况 )

关于这一问题有一个比较重要的猜想：

Pollak−Gilbert猜想： 平面上任意n点集，斯坦纳最小树长与最小生成树之长的比值的最小值是：sqrt(3)/2

由于求解斯坦纳最小树是NP问题，寻求一些近似的算法和研究斯坦纳最小树长与最小生成树之长的比值成为了重要的研究方向（ 毕竟最小生成树好求嘛- ）

斯坦纳树也有好多种类，这里我们讨论的是图上的最小斯坦纳树。

这里我引用Dreyfus，Wagner当初论文中对斯坦纳树问题的描述来更好的帮助理解图上的最小斯坦纳树。

Let a graph G be given as a pair (N,A), where N is a set of nodes, and A is a set of undirected arcs connecting nodes of N. With each arc a ∈ A is associated a positive number |a| , called the length of a. Suppose a subset Y of N is also given. The problem we consider is that of computing a subset S of A such that :
 (1) all members of Y are connected by paths composed only of arcs in S;
 (2) |S| = Σ|s| is a minimum.

图的最小斯坦纳问题最早是在1971年由Hakimi和Levin提出，并在1972年Karp证明了在一般情况下，该问题是一个NP一完全问题。

自己在求解斯坦纳最小树上的一些探索与思考

当时第一次碰到这类题目时，还不知道这时斯坦纳最小树的问题。第一反应是最小生成树，但很快反应过来它和最小生成树有个本质上的区别——最小斯坦纳树允许在给定点外增加额外的点使生成的最短网络开销最小。换句话说就是斯坦纳最小树是至少包含要求联通的点的不确定点集的最小生成树中最小的那棵树（ 好绕啊，忍不住吐槽下 ）。所以可以对这个点集进行穷举，并对每次穷举出来的点集求最小生成树，而后对答案集求Min.
这里我们假设图内有n个点，要求k个点联通，易知这样的点集有2的（n-k）次个。当时那道题目的n好像是30个，k是8个，也就是要求2^22（≈4e6）次最小生成树，这样的时间复杂度是无法接受的。（如果 n-k 小一些或许可行，就是写起来可能有些麻烦）

于是想到状态压缩DP，因为想到了TSP（ 也是图上NP问题 ）并且给出的数据都很小。然后算了下状压所有点的空间复杂度是2^30次也就是1e9，直接放弃了（可能当时心态发生了一些微妙的变化），唉，如果当时再多想一会儿的话QAQ
估计还是想不出来XD

状压所有点然后每次枚举还没加入当前联通状态下的点和加入联通的边来转移所有状态，最后再想要的状态中求Min，那道题可能还要子集DP下。（ 继续往下想的话应该就是这样了 ）不过状态压缩，状态压缩，连状态都存不下了，还咋整？

所以我们得想办法优化状态的描述，继续观察可以发现，如果我们状压所有点，那么就会出现许多无用的状态（比如不含必须点—— 题目要求联通的点|的联通状态），于是我们想到只状压要求联通的点。但其他点也可能会用到（毕竟是最小斯坦纳树，而不是最小生成树么）。所以我们再加一维表示这个联通状态中一个比较特殊的点，媒介点或者说根。（这个点很关键）。大概就是这样

接下来的部分就是1972年（大概吧）Dreyfus和Wagner发表的用于解决图的最小斯坦纳树的Dreyfus-Wagner算法！

Dreyfus-Wagner算法

网上关于斯坦纳树的资料少得可怜，还有不少是从同一个人的博客里搬来的，基本就是给个定义然后下面几行代码，好一些的就是稍微解释下转移方程其实和没说差不多，甚至我那时候还不知道这个算法的名字（ 所以我决定详细讲这部分内容 ）。

关于斯坦纳树的视频资料更是寥寥无几，我在百度上搜索就只有一个视频（附B站上视频地址），声音很轻，不过这无关紧要，可惜的是视频内容也与大多数博客相差无几，不是我想要的。后来听说2018的牛客多校赛有涉及这个算法，于是我找到了当时讲解的录屏（也是B站>.<）大概在第一场的43min左右提到了那道题。可惜视频中的讲解也只是粗略的带过。但我在这个视频中有了新的收获：李超树是李超发明的这个算法的名字是Dreyfus-Wagner！

终于花了大概一个多小时的时间，我在网上找到了当初1972年由Dreyfus和Wagner发表的论文《The Steiner Problem in Graphs》.（会放在最后的参考里）

但是这个问题的解决才刚刚开始。

进入正题：
论文开篇解释了斯坦纳树定义（图论角度），前面已经讲过了就不多赘述了。

说明了图的最小斯坦纳树问题中点与点的距离不同于与平面几何，比如平面几何中三角形的边长不能是“1”，“1”，“3”，但在图中可以，论文用“arc”代替了“edge”，也就是说图中的边是弧线。（ 论文就是严谨0.0 ）.

之后提出了一个比较重要的性质”THE OPTIMAL DECOMPOSITION PROPERTY“翻译过来就是最优分解性质（ QQ浏览器翻译的，感觉比百度翻译好用些还是TX厉害 ）。意思是任何一个斯坦纳最小树（ 这里设它是S ），以任何一个属于它的点（ 比如说P ）作为根节点，取其中的任意一棵子树（ 包括P ）设为S1，剩下的部分（ 也包括P ）设为S2，显然|S| = |S1| + |S2|,(|S|代表S的值)那么S1,S2都是各自点集的斯坦纳最小树。（ 是不是有点像DP里的最优子结构的性质，从这里可以看出有些DP的意思其实很明显了。 ）

论文中证明过程写在附录中

这个证明其实很简单，一般DP里面证明最优子结构都是用的反证法。假设对于S1中关键点集的斯坦纳树中有比S1更小的设为T，那么就有 |T| + |S2| < |S1| + |S2| = |S|,
这显然与S是最小斯坦纳树矛盾。

至此一些必要的解释工作就完成了

Dreyfus和Wagner提出了他们的解法

根据最优分解性质，要求得S的最小斯坦纳树长，需要枚举点p ∈ n作为拆解的媒介点，同时还要对每个p点枚举子树（实际程序就是枚举子集）。最后把每种情况下得到的答案取Min
。但在此之前，得预先处理出其所有子结构（子集）的解。很明显这是一个递归的过程。

因此Dreyfus和Wagner提出以下解法

Y是要求联通的点集，||Y||是点集元素数，

（1）Remove one node, q, from Y. Call the set Y - {q} set C.
（2）Solve Steiner problems connecting each set of 2 nodes of C and 1 node n ∈ N. (n can be a member of C, or it can even be node q itself).
（3）Use this result to solve Steher problems connect- ing each set of 3 nodes of C and 1 node n ∈ N.
……
（||Y|| - 2）Solve Steiner problems connecting each set of ||Y|| - 2 nodes of C and 1 node n ∈ N.
（||Y|| - 1）Solve the Steiner problem connecting q and set C.

具体程序的枚举顺序可能有所不同，但实际上是没有影响的。

斯坦纳森林

先为所有集合中的点做斯坦纳树，求出f[S][i]。
设g[C]表示集合联通状态为C的最小代价。
先将每个集合的g[C]赋初值
g[C] = min(f[C][1~n])
然后做子集DP。加个check检查下确保每个子集都是合法的（做到题目就知道了）。
